# Folder and Graphs Plus

Folder and Graphs Plus extends Folder2Graph-style folder nodes in Obsidian's Graph View. It keeps the changes visual: vault files, folders, and graph configuration are not renamed, moved, merged, or rewritten.

The plugin can:

- Colour folder nodes from configured folder colour rules.
- Colour graph lines that touch matching folders.
- Optionally combine same-name folders in Graph View.
- Show folder emoji markers for folder nodes.
- Bold exact configured folder labels.
- Add extra attachment and folder filters into the native Graph View filter panel.

## Installation

Install from Obsidian's Community plugins browser once the plugin is available there.

For manual installation, download `main.js`, `manifest.json`, and `styles.css` from a GitHub release and place them in:

```text
<vault>/.obsidian/plugins/folderandgraphs-plus/
```

Then enable Folder and Graphs Plus from Obsidian's Community plugins settings.

## Usage

Open the plugin settings to add folder colour rules. Each rule can set the folder colour, choose whether child notes inherit that colour, toggle graph line colouring, and bold the exact matching folder label.

Use `Show folder emoji in graph` to prefix every Folder2Graph folder node label with `📁`. Use Obsidian's native Graph View filter panel for the plugin's attachment, folder-node, and canvas filters.
